Work based on day 31 prompt of Genuary 2026.

Prompt: *GLSL day. Create an artwork using only shaders.*

Being inspired by the prompt, the Hyperobjects tooling was extended with a chromatic abberation effect implemented as post-processing shader in the rendering pipeline.

The scene itself consists of a rotating cube, where at random, the width and height of the cube will randomly be reduced. Technically then each pixel in the work is generated by a shader altough the scene itself was configured at the layer of compiled Swift code which then passes the cube data per frame towards the shader pipeline.  So the full work is created with more than just shaders.
